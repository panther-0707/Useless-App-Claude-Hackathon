# Useless-App-Claude-Hackathon

A professional bank landing page built with React and Vite, featuring an AI-powered chatbot.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file in the root directory (copy from `.env.example`):
```bash
ANTHROPIC_API_KEY=your_api_key_here
PORT=3001
```

3. Run the backend server (in one terminal):
```bash
npm run server
```

4. Run the frontend development server (in another terminal):
```bash
npm run dev
```

5. Build for production (static files):
```bash
npm run build
```

The static files will be in the `dist` folder after building.

## Features

- Professional bank landing page
- 6 service cards linking to feature pages
- **AI-Powered Chatbot**: Ragebait AI on the Contact page - an extremely unhelpful but hilarious chatbot powered by Claude
- Fully responsive design
- Built with React and Vite for static site generation

## AI Chatbot

The Contact page features a unique AI chatbot called "Ragebait AI" that provides entertaining, intentionally unhelpful responses to user questions. It's powered by Anthropic's Claude API and designed to give spicy, brain-rot hot takes instead of useful answers - perfect for a comedic banking experience!
