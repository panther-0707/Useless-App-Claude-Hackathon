# Useless-App-Claude-Hackathon

A professional bank landing page built with React and Vite.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production (static files):
```bash
npm run build
```

The static files will be in the `dist` folder after building.

## Features

- Professional bank landing page
- 6 service cards linking to feature pages
- Fully responsive design
- Built with React and Vite for static site generation
